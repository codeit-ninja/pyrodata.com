[edit] Place for Materials?

Hi Pudi. Just wanted to say that my PM is obsolete, I managed to join the group.

Regarding the fact that a sidebar cat for materials such as paper, string, etc. is still missing, I wanted to propose that we make one. In contrase, the binders cat is well covered by chemicals and we could simply move/recategorize the articles. What do you think?

-

Admiral, I don't know if it had any effect that I went into the forum controlpanel and changed your group, don't know if I did it right, forgot to write back. Actually I think a better option is to make only one category in the sidebar called chemicals, and then make several subcategories like is done here: http://pyroforum.nl/wiki/index.php/Categorie:Chemicali%C3%ABn

I don't like the sidebar to big, I would much prefer the way it's done in the link, I hope you follow me. Also I have noticed being a lot of short articles covering parts of the same thing, like an article called 'Kraft paper' and a 'Kraft Paper Weights' which would be at least a thousand times easier if the weights were just listed in the Kraft paper' article.

/Pudi

--

Thanks for trying to help me join the group, but I´ve already managed joining it. You also did a good work collapsing the two pages about paper. Concerning sidebar categorization, I´m not sure if splitting up chemicals into binders, oxidizers etc. is necessary at all. I never got why there is a separate "binders" category. The same is true with "fuses".

When it comes to categorization, I like the way pertinent literature discerns things. E.g. Shimizu sets an overall category called "materials", listing chemicals, papers, black powder grades, fuses, string and core materials under it. This is a good practice since all these things indeed are materials used in fireworks and doing so would simplify side categorization by eliminating categories. It would also resolve the current issue where to put important information about paper and string etc. However the exact layout of realisation should be considered if we decided to make it because it would be chaotic if all these things were listed mixed up like in the case of your link. We should probably also consider if the "shell tutorials" cat needs to stay separate from other tutorials.

However we decide to make it - we will have to create some room for things like string, rice hulls etc., materials as important as perc when it comes to fireworks assembly. By the way, looking forward to some nice months of cooperations, mate. --AdmiralDonSnider 09:58, 23 January 2009 (EST)
